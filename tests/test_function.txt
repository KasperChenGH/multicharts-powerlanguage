{ A PL function returns its value by assigning to its own name. }

Inputs:
    Len(Numeric);

Variables:
    avgRng(0);

avgRng = Average(Range, Len);

If avgRng <> 0 Then
    RangeRatio = Range / avgRng
Else
    RangeRatio = 0;
