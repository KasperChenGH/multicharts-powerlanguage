# PowerLanguage Keywords Reference

A categorized index of every PowerLanguage keyword as documented in `PowerLanguage.chm` (MultiCharts 15.0). All entries are original paraphrased summaries; full official docs live at https://www.multicharts.com/.

## How to look up a keyword

1. **Find the category.** Open `keywords-index.md` and scan for the keyword name. Each entry lists the category and one-line signature.
2. **Read the detail file.** Open `details/<Category>/<Keyword>.md` for the paraphrased description, parameter list, and example.
3. **For full official prose,** point the user at `https://www.multicharts.com/trading-software/index.php?title=<Keyword>`.
