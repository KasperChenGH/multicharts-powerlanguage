# ARW_New_Self_BN

**Category:** Arrow_Drawing
**Signature:** `ARW_New_Self_BN(BarNumber, PriceValue, Direction)`

The same as `Arw_New_BN`, except the arrow is displayed on the SubChart containing the study (rather than the price chart).

**Parameters**
- `BarNumber` *(numeric, required)* — see official docs
- `PriceValue` *(numeric, required)* — see official docs
- `Direction` *(truefalse, required)* — see official docs

**Example (illustrative)**
```
Value1 = ARW_New_Self_BN( 14, Close, True );
```
