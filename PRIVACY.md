# Privacy Policy

**Plugin:** multicharts-powerlanguage
**Last updated:** 2026-06-10

## Summary

This plugin does not collect, store, transmit, or share any user data.

## Details

- The plugin consists entirely of static skill files (Markdown documentation) that are loaded locally by Claude Code. It contains no executable code, no MCP servers, no hooks, and no scripts.
- It makes no network requests. Reference links inside the skills point to the official MultiCharts wiki and TradingView documentation; these are only fetched if you or Claude explicitly choose to open them.
- No telemetry, analytics, or usage tracking of any kind is included.
- Any code you write or convert using these skills stays in your Claude Code session and is governed by Anthropic's privacy policy, not by this plugin.
